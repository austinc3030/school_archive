2. Changed SSID and Wi-Fi Password

Some router features are rarely used and may pose a threat to your network. Turning off features such as WPS will make your network more secure.

Making sure to use WPA2 saves your network from known vulnerabilities compared to other encryption methods.

Frequently Asked Questions

What is a router?

Routers are networking devices which allow multiple devices to share an Internet connection. The router sits between two networks and routes data appropriatly between the two. Usually, one network would be public Internet and the other network would be your home (Local Area Network or LAN).

What is the difference between Router and a Modem?

The router is not responsible for talking to your Internet Service Provider, the modem handles that. Different Internet connections such as cable, dial-up, satellite, optical, etc... require different types of modems. While different modems vary depending on the type of Internet connection, they all have an Ethernet port on the back end which serves as input for the router. This way any router may work with any Internet connection.

Why should I update my Firmware?

Typically, a new router will be running older firmware (firmware is the operating system inside the router). Router manufacters may release updates to the router's firmware, which close known vulnerabilites and fix bugs within the equipment. Keeping the router updated is a great way to make sure your home network is as secure as it can be.

Why change the default passwords?

Default passwords are a large issue for routers and really shouldn't be allowed. Even default passwords that appear random are not actually so. Eventually, someone will discover a formula for creating those default passwords and may use that information plus public information from the router to discover all necessary passwords.
