Name/Password Protect Wi-Fi

Consider changing the name of the network (This is called the SSID). There is a default name that is already chosen, but there are advantages when using a different name.

The default SSID makes it easier for hackers to break into your encryption (we will come to this later). This is due to the default network name being part of the encryption algorithm and password cracking tables (Rainbow Tables) which include these popular default network names.
Your network name should not give away any personal information. For example, "The Smith Family" is poor choice if your last name is Smith. If you live in apartment 7B, name your network "Apartment 9A". Don't make it easy for attackers to target you.
Just like before, a strong password makes it significantly more difficult for hackers to break into your system. Strong passwords are considered over 8 characters in length and comprise of letters, numbers and symbols. They contain letters in both uppercase and lowercase.
